单变量线性回归

定义

  • 由训练样例(training example)组成的集合就是训练集(training set)，如图所示，其中(x,y)就是一个训练样例，(x(i),y(i))就是第i个训练样例
  • 一般地，令D = {x1,x2,…,xm}表示包含m个示例的数据集，每个示例由d个属性描述，则每个示例xi = (xi1;xi2;…;xid)是d维样本空间X中的一个向量，其中xij是xi在第j个属性上的取值
  • 特别的，当属性的数量为1时，称之为单变量线性回归

假设函数(Hypothesis Function)-hθ(x)

  • 使用某种学习算法对训练集的数据进行训练，我们可以得到假设函数，即特征同标记之间的某种关系
  • 在房价的例子中，假设函数就是一个房价关于房子面积的函数学习算法的目的就是为了得到这个函数，这样在这之后，给定一个房子的面积，我们就可以对其房价进行预测了

代价函数(cost function)-J(θ0,θ1)

  • 只要我们知道了假设函数，就可以进行预测了.对于假设函数中的两个未知的量θ0和θ1，当选择不同的θ0和θ1时，那么模型的泛化能力肯定也不一样，如图所示，列举了三种不同的θ0和θ1下的假设函数
  • 为了选择泛化能力更强的θ0和θ1组合，我们选择使用均方误差来作为衡量标准，即我们想要每个样例的估计值与真实值之间的的差的平方的均值最小
  • 估计值同真实值之间的均方误差即为代价函数(cost function)
  • 其中，代价函数的最小值所对应的θ0和θ1即为对模型的优化目标

假设函数与代价函数

  • 首先，为了更方便的研究hθ与J(θ0,θ1)的关系，先令θ0 = 0，得到了简化的假设函数，由假设函数的定义可知此时的假设函数是经过原点的直线，相应的也可以求得简化的代价函数
  • 简化之后，我们令θ1 = 1，就得到hθ(x) = x,其中图中的红叉代表训练样例，通过代价函数我们可以计算出当θ0 = 0，θ1 = 1时，J(0,1) = 0，对应右图中的(1,0)坐标
  • 重复上面的步骤，分别计算θ1不同取值时的代价函数值，将这些点连接起来就可以得到J(0,θ1)关于θ1的函数图像，我们的目标是找到一个θ使得假设函数的代价函数最小，从图像中我们可以得出，当θ1 = 1时，J(θ)取得最小值的结论
  • 当我们不令θ0 = 0时，代价函数的图像会发生很大的变化，当我们同时设置θ0和θ1的值时，然后再绘出J(θ0,θ1)的图形，因为此时有两个变量，所以J(θ0,θ1)是一个曲面

梯度下降法

  • 在向量微积分中，标量场的梯度是一个向量场。标量场中某一点的梯度指向某一点的梯度指向在这点标量场增长最快的方向（固定方向的长度），梯度的绝对值是长度为1的方向重函数最大的增加率，特别的一元函数的梯度就是导数
  • 梯度下降法(Gradient descent)是一个一阶最优化算法，通常也称为最速下降法。要使用梯度下降法找到一个函数的局部最小值，必须向函数上当前点对应梯度（或者近似梯度）的反方向的规定步长距离点进行迭代搜索。如果相反地向梯度正方向迭代进行搜索，则会接近函数的局部极大值点（这个过程被称为梯度上升法）
  • 可以将梯度下降法和下山联系起来，如果想要尽可能快的下坡，应该每次都往坡度最大的方向下山
  • 梯度下降法得到的结果同初始状态有关，当从不同的点开始时，可能会到达不同的局部极小值（陷入局部最优解）
  • 局部最优解，是因为不同的起始点导致的算法进入不同的局部最低点
  • 但是对于线性回归的代价函数总是一个凸函数(Convex Function)，这样的函数没有局部最优解，只有全局最优解，所以我们在使用梯度下降法的时候，总会得到一个全局最优解

梯度与学习率

  • 为了研究梯度下降法是如何帮助我们找到最优解的，我们设θ0等于0，假设一开始选取的θ1在最低点的右侧,此时的梯度(斜率)是一个正数，从图中我们可以知道，当使用算法更新θ1的时候，它的值会减小，慢慢靠近最低点
  • 类似的假设一开始选取的θ1在最低点的左侧，此时的梯度是一个负数，根据上面的算法更新θ1的时候，它的值会增大，也就是靠近最低点
  • 如果一开始选取的θ1恰好在最适位置，那么更新θ1时，它的值不会发生变化
  • 学习率α直接影响每一次梯度下降的幅度，如果α太小，那么0的值每次变化的也会很小，那么梯度下降的就会非常慢，相应的如果α太大，那么0的值每次变化的会很大，梯度下降的就会非常快
  • 梯度下降的过慢，会导致算法更新所需要的时间非常多，但是梯度下降的过快，有可能导致在梯度的下降的过程中，越过了最低点，甚至永远没有办法到达最低点（发散）
  • 随着越来越接近最低点，相应的梯度（绝对值）也会逐渐减小，所以每次下降的幅度也会逐渐减小，所以我们不需要取改变学习率α的值
