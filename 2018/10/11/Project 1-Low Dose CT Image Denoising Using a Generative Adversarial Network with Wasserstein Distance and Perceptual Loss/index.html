Project 1 - Low Dose CT Image Denoising Using a Generative Adversarial Network with Wasserstein Distance and Perceptual Loss

Posted on 2018-10-11

Low Dost CT Image Denoising Using a Generative Adversarial Network with Wasserstein Distance and Perceptual Loss

  • 使用具有EM距离和损失感知的生成对抗网络进行CT图像降噪

摘要

  • CT的不断发展和在医疗实践中的广发应用，引起了关于放射性物质剂量对患者的影响的广泛关注。减少放射性物质剂量可能会导致噪声和误差的增加，而这些会极大的影响放射线研究者的判断和置信度。因此，诊断的表现需要由先进的根据低剂量CT数据的图像重建方法来提高，但是由于其病态的性质，这是一项具有挑战的任务。在过去的很多年里，许多低剂量CT图像重建方法得到了许多重大的成果。然而，大多数针对该应用开发的算法，包括最近普及的深度学习技术，旨在最小化去噪CT图像与一般惩罚下的正确标注的样例之间的均方误差。在积极的降噪之后，尽管信噪比峰值得到了改善，但是均方误差或者加权均方误差的方法可能会危害重要结构细节的能见度。

  • 使用MSE或weighted-MSE-based方法，具有一定的缺陷

  • 本论文介绍了一种利用EM距离和感知相似度生成对抗网络（GAN）的新型CT图像降噪方法。EM距离是最优传输理论的核心概念，并且可以改善GAN的性能。感知损失通过比较已建立的特征空间中去躁输出的感知特征与正确标注的样例的感知特征来抑制噪声，而GAN更倾向于将噪声统计数据分布从强到弱的进行迁移。因此，我们所提出的方法在降低噪声等级的同时，还尽可能保留了图像的关键信息，它将我们的视觉感知知识转移到图像去噪任务。在利用医疗CT图像的实验中，我们得到了令人瞩目的结果。

介绍

  • X射线计算机断层扫描技术是现代医院与诊疗机构中最重要的成像方式之一。然而，对于患者而言存在潜在的放射性风险，由于x射线可能会造成基因损伤和辐射剂量可能会增加患癌的可能性。降低放射性物质的剂量会增加图像重建中的噪声和误差，进而对诊断信息造成影响。因此，许多人为设计更好的图像重建方法或低剂量CT图像处理方法做出了很多努力。这些方法，通常归为三类：（1）重建前正弦图过滤（2）迭代重建（3）重建后图像处理
  • 过去数十年间，研究人员致力于开发用于低剂量CT图像的迭代算法。通常，那些算法优化了包含准确的系统模型、噪声统计模型和图像域中的先验信息的目标函数。主流图像先验包括总变异和其变体，以及字典学习。这些迭代重建算法极大地提高了图像的质量但是它们仍然损失一些细节并受到残留误差的干扰。并且，它们需要很大的计算代价，这也成为了其实际应用的瓶颈。
  • 另一方面，正弦图预过滤和图像后处理较迭代重建具有计算效率高的优势。对于正弦图域过滤，噪声特性在正弦图域中很好的建模。然而，用户不容易获得商用扫描仪的正弦图数据，并且这些方法可能会存在解析度损失和边界模糊的问题。正弦数据需要精确的处理，否则重建图像可能会存在误差。
  • 同正弦图过滤降噪不同，图像后处理直接作用于图像。在图像领域许多研究者为降低低剂量CT图像噪音和抑制误差做出了巨大努力。例如，非局部均值法就适用于CT图像降噪。受压缩感知方法的启发，提出了一种改进的K-SVD方法来减少CT图像中的伪影。在某些CT图像任务中，块匹配3D算法被应用于图像恢复。利用这种图像后处理方法，图像的质量很明显得到改善，但是在处理的图像中经常可以观察到过渡平滑和残留误差。考虑到CT图像噪声的不均匀分布，这些问题难以解决。
