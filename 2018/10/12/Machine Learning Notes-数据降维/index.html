数据降维

数据降维

  • 对于如下图所示的二维特征，我们可以找到一条支线，将所有的点映射到这条直线上，这样就将二维的数据降到一维，得到一个新的特征$z_1$，降维不仅可以让我们节省计算资源，更重要的是可以让学习算法运行的更快

  • 同样的，我们还可以将三维数据降到二维数据

主成分分析

概述

  • 主成分分析(PCA, Principal Components Analysis)是用来对数据降维的非监督学习算法.假设我们有如下图所示的数据，我们希望将数据降到一维，那么PCA是如何找到那条合适的直线？

  • 如图所示.当向正斜率的直线投影时，每个样本的投影误差都很小.但是，当向负斜率的直线投影时，每个样本的投影误差都很大

  • 由此，我们得到PCA的优化目的为，找到某些向量，使得每个样本对这些向量的投影距离（投射误差）和最小

  • 通过上面的内容也许会让我们想到线性回归，但PCA和线性回归是完全不同的两个算法.在线性回归中（下左图），我们想要的是能够拟合数据的一条支线，最小化的是亮点之间y的差；而在PCA中（下右图）我们最小化的是点到直线的距离（注意下右图中点垂直于线的距离）.并且，在线性回归中，有一个标签y；而在PCA中所有的都是特征

PCA算法

  • 在使用PCA之前，我们需要对数据进行特征缩放(feature scaling)或均值归一(mean normalization)

  • 在PCA中，我们需要计算的就是向量u和新的特征z

  • 首先，我们需要计算出矩阵$\Sigma$，然后使用奇异值分解(sigular value decomposition)来计算[U, S, V].我们需要的是n x n的矩阵U，如果我们需要将数据从n维降到k维，取U的前k列，记为$U_{reduce}$

  • 然后，根据矩阵乘法的原则，我们就可以将x降到k维，也就得到了向量z

PCA总结

  • 使用PCA的作用之一便是方便进行数据可视化

  • PCA不能被用来解决过拟合的问题

  • 在使用PCA之前应该考虑先使用原始数据，如果使用原始数据不能达到效果，再考虑使用PCA
